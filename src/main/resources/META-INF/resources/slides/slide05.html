Orchestrierung mit Nomad

Konfiguration ohne YAML

  • Kein rein deklarativer Ansatz
  • Es wird die hauseigene Sprache HCL verwendet
  • Bietet Logikfunktionen und Operatoren
  • company = "adesso"
message = "Hallo "
loud_message = upper(message)
colors = [ "red", "blue" ]
options = {
  color: element(colors, 1),
  amount: 100
}
configuration {
  service "greeter" {
    message = loud_message
    options = var.override_options ? var.override_options : var.options
  }
}
  • Nomad Pack bietet Helm-ähnliche Funktionalität
